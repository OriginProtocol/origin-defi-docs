# 공급 탄력성

## 슈퍼 파워\(Super Powers\)

슈퍼 히어로가 되는 과정은 상당히 단순합니:

```
$ give me super-powers
```

{% hint style="info" %}
 슈퍼파워는 랜덤으로 부여됩니다. 만약, 마음에 들지 않을 경우 이슈를 제출해 주세요.
{% endhint %}
